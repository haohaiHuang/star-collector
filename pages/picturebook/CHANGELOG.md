# Star Collector - 功能描述文档

## 项目概述

**Star Collector** 是一款专为移动端设计的星星收集激励应用，帮助用户通过设定和达成每日/每月星星目标来培养好习惯。

**技术栈：** HTML5 + Tailwind CSS + Vanilla JavaScript
**部署平台：** GitHub Pages
**数据存储：** localStorage（浏览器本地存储）

#### 里程碑功能说明

1. **进度条里程碑节点**
   - 进度条中添加25%、50%、75%里程碑节点
   - 每个节点为 w-1.5 h-1.5 rounded-full 圆形标记
   - 节点下方显示对应百分比标签

2. **里程碑弹窗**
   - 标题显示 "Milestone Reached!"
   - 内容显示 "You've reached the XX% milestone! (XX/XX stars)"
   - 包含星星装饰图标
   - 包含 "Keep Going" 按钮关闭弹窗

3. **里程碑触发逻辑**
   - 使用 `Math.round()` 计算百分比
   - 当进度超过里程碑百分比时触发（如49%→51%触发50%里程碑）
   - 每个里程碑只触发一次
   - 里程碑状态持久化存储

#### 打卡日历功能说明

1. **日历入口**
   - 主界面左上角添加日历按钮
   - 点击按钮弹出打卡日历弹窗

2. **日历显示**
   - 显示当月所有日期，今日高亮
   - 打卡日期下方显示星星图标
   - 显示本月打卡天数和连续打卡天数

3. **打卡记录**
   - 每次添加星星时记录当天打卡
   - 重新开始累积时打卡记录清零
   - 修改目标时打卡记录保留
   - 扣减星星时从当天最近记录开始删减

#### 奖励系统功能说明

1. **奖励设置**
   - 在设置界面添加"Reward Settings"下拉框
   - 提供预设奖励选项（Movie Night、Ice Cream、New Book、Game Time、Special Treat）
   - 支持选择"Custom..."后手工输入自定义奖励
   - 点击"Save Changes"保存设置

2. **奖励领取**
   - 达到目标后，点击"Claim Reward"按钮
   - 弹出奖励领取弹窗，显示设置的奖励内容
   - 点击"Collect Reward"完成领取
   - 领取后自动重置星星累积，开始新的累积状态

#### 核心逻辑

1. **添加星星**
   - 点击按钮 → `totalStars++` → `monthlyStars++` → 保存 → 更新UI → 检查里程碑

2. **里程碑检测**
   - 每次添加星星后检查进度是否达到里程碑
   - 使用 `Math.round()` 计算百分比
   - 当进度超过里程碑百分比时触发（如49%→51%触发50%里程碑）

3. **月度重置检测**
   - 每次页面加载时比较 `lastMonth` 与当前月份
   - 如果不同 → 重置 `monthlyStars = 0` → 更新 `lastMonth`

4. **目标达成检测**
   - 每次添加星星后检查 `totalStars >= goal`
   - 达成时显示祝贺弹窗，进度条添加发光效果

5. **重置逻辑**
   - `totalStars = 0`
   - `monthlyStars = 0`
   - `lastMonth = 当前月份`
   - 隐藏弹窗

#### 设计规范

- **配色**：琥珀色/金色暖色调（遵循 Luminous Achievement Framework）
- **字体**：Plus Jakarta Sans（标题）+ Be Vietnam Pro（正文）
- **图标**：Material Symbols Outlined
- **圆角**：1rem / 2rem / 3rem / full (9999px)
- **无 1px 实线边框**：使用色调差异区分区域
